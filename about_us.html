How to install

Installation from repository

Fetch and install the GnuPG key

wget -O - http://pkg.yeti-switch.org/keys/yeti-switch.gpg.key | apt-key add -

add these line to your /etc/apt/sources.list

deb http://pkg.yeti-switch.org/debian wheezy main
