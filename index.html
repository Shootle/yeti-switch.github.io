--- layout: main title: YETI - VOIP for humans ---
YETI is open source project based on SBC module of SEMS. The main goal is to move routing logic and entire softswitch behaviour to separated layer. This approach provides flexibility, ability to quickly add a non-trivial features and simply integrate with other systems. It is possible to completely change the business logic changing only management layer component.
YETI consist of few separate parts:
  • Core - modified SEMS and YETI SEMS module binaries
  • Management layer - set of tables and appropriate PL/pgSQL stored procedures. As a database system we have chosen PostgreSQL.
  • Web-interface to manage YETI on Rails
