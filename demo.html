Hint!

Use disconnect code as number to call. For instance, if you dial 200, call will be connected successfully and you will hear voice. If you dial 503, YETI will reject call with 503 disconnect code. Supported codes: 200, 403, 404, 486, 503, 603 After finishing your call, you can check CDR in WEB console
